CSS Specificity

Specificity คืออะไร

เป็นหลักการที่ใช้ในการควบคุมการทำงานของ CSS โดยขึ้นกับ selector ที่ถูกนำมาใช้กำกับให้เกิดคุณสมบัติต่อ element ต่างๆ

Specificity มีหลักการอย่างไร

selector แต่ละชนิดจะมีค่าที่แสดงถึงลำดับความสำคัญอยู่ในลักษณะเชิงปริมาณ กล่าวคือค่าเหล่านี้สามารถรวมกันได้ และเมื่อผลรวมสุดท้ายในกรณีที่มี selector ที่ชี้ถึง element เดียวกันฝั่งไหนมากกว่าจะได้รับการแสดงผลไป

ตารางค่าลำดับความสำคัญจากการประมาณของ selector
ชนิดของ selector ค่าที่ประมาณให้
Inline Style 1000 (มากที่สุด)
ID 100
Class 10
Element 1
Chain Selector ผลรวมทั้งหมดของ selector
!important บังคับใช้ไม่ขึ้นกับการเปรียบเทียบ
